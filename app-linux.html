Digitalku :: Tutorial Instal Aplikasi

Panduan singkat untuk menginstal browser di VPS menggunakan Terminal. Tutorial ini direkomendasikan untuk VPS berbasis Linux.

Cara mencari Aplikasi

  1. Buka Terminal pada Desktop VPS.
  2. Jalankan perintah berikut:
apt search (nama aplikasi)
apt install (nama aplikasi)

Cara Instal Chromium

  1. Buka Terminal pada Desktop VPS.
  2. Jalankan perintah berikut:
apt install chromium

Cara Instal Firefox

  1. Buka Terminal pada Desktop VPS.
  2. Jalankan perintah berikut:
apt install firefox

Pastikan VPS terhubung ke internet sebelum menjalankan perintah instalasi.

Jika browser tidak digunakan untuk trading, disarankan untuk menutupnya agar performa VPS tetap optimal.
