Digitalku :: Instal Aplikasi Windows

Walaupun DigiOS berbasis Linux, sistem ini telah dioptimalkan untuk menjalankan aplikasi Windows dengan sangat baik, khususnya aplikasi trading seperti MetaTrader.

Cara Instal Aplikasi Windows

  1. Download aplikasi Windows dengan ekstensi .exe.
  2. Buka file hasil download di Desktop atau folder Downloads.
  3. Lakukan double klik atau klik kanan → Open pada file .exe.
  4. Ikuti proses instalasi seperti pada Windows pada umumnya.
  5. Setelah selesai, aplikasi siap digunakan.

Cara Uninstall Aplikasi Windows

  1. Temukan aplikasi Windows Uninstaller di Desktop.
  2. Double klik Windows Uninstaller.
  3. Pilih aplikasi Windows yang ingin dihapus.
  4. Klik uninstall dan tunggu hingga proses selesai.
