Digitalku :: Windows 7 Reborn

Windows 7 tetap menjadi primadona sistem operasi VPS Forex saat ini karena kecepatannya yang sangat baik dan juga stabil, dan kabar baiknya, aplikasi terbaru juga masih dapat dijalankan di VPS Windows.

Persyaratan Sistem

  • VPS Menggunakan OS Windows 7
  • VPS yang diaktivasi diatas 04 Maret 2025 sudah terinstal Fitur ini
  • VPS yang diaktivasi sebelum tanggal tersebut bisa menghubungi CS Digitalku untuk pemasangan

Cara Penggunaan

  • Klik kanan Properties pada Aplikasi yang akan dijalankan
  • Pilih Tab VxKex
  • Centang Enable VxKex for this program dan Report a different version of Windows
  • lalu Pilih Windows 11 lalu OK

Lakukan setting ini hanya untuk pertama kali saja setiap aplikasi. selanjutnya bisa langsung dibuka seperti biasanya
